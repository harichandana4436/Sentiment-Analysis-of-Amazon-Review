## 📝 Sentiment Analysis on Amazon Reviews

### 📌 Project Overview
This project demonstrates the implementation of a Machine Learning based Sentiment Analysis system using Amazon product reviews. The application predicts the sentiment of customer reviews as Positive, Neutral, or Negative using Natural Language Processing (NLP) and Machine Learning techniques.

The project highlights real-world text analysis operations such as:
- Text preprocessing
- Feature extraction using TF-IDF
- Machine Learning model training
- Sentiment prediction using user input
- Interactive web application deployment using Streamlit

## ▶️ How to Run the Project

- Install required libraries

          pip install -r requirements.txt

- Run the Streamlit application

         streamlit run app.py
- Open the local URL shown in the terminal
- Enter Amazon reviews and view sentiment predictions
